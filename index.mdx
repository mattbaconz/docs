## What is SeasonManager?

**SeasonManager** is a comprehensive plugin designed to add seasonal mechanics to your Minecraft server. It allows you to create dynamic, evolving worlds with scheduled events, cross-season progression, and competitive leaderboards.

Whether you're running a survival server, an SMP, or a competitive game mode, SeasonManager provides the tools to keep your gameplay fresh and engaging.

## Editions

SeasonManager is available in two editions:

- **Core**: The free version with essential features for seasonal management.
- **Prime**: The premium version with advanced features like the visual dashboard, cross-season stash, and event hooks.
