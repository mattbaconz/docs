Installing SeasonManager is straightforward. Follow these steps to get started.

## Requirements

*   **Java**: Java 21 or higher.
*   **Server Software**: Paper, Spigot, or any Folia-supported fork (1.20+).
*   **PlaceholderAPI** (Optional but recommended): For using placeholders in messages and scoreboards.

## Installation Steps

1.  **Download** the latest version of SeasonManager (Lite or Prime) from the official source.
2.  **Stop** your Minecraft server.
3.  **Place** the downloaded `.jar` file into your server's `plugins` folder.
4.  **Start** your server.
5.  **Verify** the installation by checking the console for SeasonManager startup messages.

## First Run

Upon the first run, SeasonManager will generate a default `config.yml` file. You can edit this file to customize the season duration, notifications, and other settings.

See the [Configuration](/seasonmanager/configuration) page for more details.
