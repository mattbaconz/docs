description: 'Command Reference'
---

All commands are prefixed with `/pyramid` or `/py`.

| Command | Description | Permission |
| :--- | :--- | :--- |
| `/pyramid invite <player>` | Send an invitation to a player to join your organization. | `pyramid.use` |
| `/pyramid join <player>` | Accept an invitation. | `pyramid.use` |
| `/pyramid gui` | Open the Dashboard to view performance and downline. | `pyramid.use` |
| `/pyramid takeover <player>` | **(Stonkz Only)** Acquire a player if you satisfy the shareholding requirements. | `pyramid.use` |
| `/pyramid help` | View the help menu. | `pyramid.use` |
| `/pyramid reload` | Reload the configuration (Admin). | `pyramid.admin` |
| `/pyramid setupline <player> <target>` | Manually set a player's upline (Admin). | `pyramid.admin` |
| `/pyramid reset <player>` | Reset a player's data (Admin). | `pyramid.admin` |
