## Requirements

- **Java**: Java 21+
- **Server**: Paper, Folia, or Purpur (1.20+)
- **Vault**: The standard economy API.
- **Economy Plugin**: Any Vault-compatible economy plugin (EssentialsX, etc.).

## Installation Steps

1.  Download `Pyramid.jar`.
2.  Drop it into your `/plugins` folder.
3.  Restart your server.
4.  Configure `config.yml` to your liking.
5.  (Optional) Install [Stonkz](https://modrinth.com/plugin/stonkz) for advanced features.
