---
title: 'Permissions'
description: 'Permission Reference'
---

| Permission | Description | Default |
| :--- | :--- | :--- |
| `pyramid.use` | Allows access to standard commands (`invite`, `join`, `gui`). | `true` |
| `pyramid.admin` | Allows access to administrative commands (`reload`, `setupline`, `reset`). | `op` |
| `pyramid.bypass` | Players with this permission are exempt from commission deductions (in Tax mode). | `false` |
